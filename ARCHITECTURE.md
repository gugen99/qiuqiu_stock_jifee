# 股票咨询收费平台架构

## 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                      用户端 (微信/浏览器)                         │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  阿里云服务器 (前台)                                             │
│  ├── Next.js 14 前端 (收费/计费界面)                             │
│  ├── 微信支付集成                                                │
│  ├── 微信登录/认证                                               │
│  ├── 订阅管理 + PDF 生成推送                                      │
│  └── 与本地 Bot 通信的 API 网关                                   │
└─────────────────────────────────────────────────────────────────┘
                              │ WebSocket / HTTP
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  本地服务器 (股票分析核心)                                        │
│  ├── OpenClaw 龙虾机器人 (后台客服)                              │
│  ├── 股票数据分析引擎                                            │
│  ├── 实时行情接入                                                │
│  └── 咨询回复生成                                                │
└─────────────────────────────────────────────────────────────────┘
```

## 目录结构

```
stock-advisor-platform/
├── frontend/          # 阿里云部署 - 前台收费系统
├── local-bot/         # 本地部署 - 股票分析机器人
├── shared/            # 共享类型和工具
└── docker-compose.yml # 一键部署
```

## 核心功能

1. **微信生态集成**: 登录、支付、消息推送
2. **订阅分级**: 免费/基础/高级(PDF报告)
3. **本地Bot通信**: WebSocket 实时咨询
4. **PDF生成**: 高端订阅用户的研报推送
